PostgreSQL database, authentication, file storage, edge functions

Verify These Are Complete

  1. Open supabase.com/dashboard and sign in to your project talent-stlucia

2. Cloudflare (DNS, CDN, Security)

Free CDN, SSL management, DDoS protection, caching, edge rules

Set Up Now
⏱ ~15 minutes + up to 24 hours for DNS propagation
  1. Go to dash.cloudflare.com/sign-up and create a free account
  2. Click "Add a site" and enter stlucia.studio (the parent domain, not the subdomain)
  3. Select the Free plan and click Continue
  4. Cloudflare will scan your existing DNS records. Review them and click Continue. It will show you two nameservers (e.g., ada.ns.cloudflare.com and bob.ns.cloudflare.com)
  5. Go to your domain registrar (wherever you bought stlucia.studio) and replace the current nameservers with Cloudflare's two nameservers. Save.
  6. Back in Cloudflare, click "Check nameservers". It may take a few hours to propagate. You'll get an email when it's active.
  7. Once active, go to DNS → Records and add/verify this CNAME:
    Type: CNAME Name: talent Target: ibak-app.github.io Proxy status: Proxied (orange cloud ON)
  8. Go to SSL/TLS and set encryption mode to Full
  9. Go to Speed → Optimization and enable Brotli compression
  10. Go to Caching → Configuration and set Browser Cache TTL to 1 month
  11. Go to Rules → Page Rules and add:
    Rule 1: talent.stlucia.studio/assets/* Cache Level: Cache Everything Edge Cache TTL: 1 month Rule 2: talent.stlucia.studio/* Always Use HTTPS
What you get: Free global CDN, automatic SSL renewal, DDoS protection, Brotli compression, browser caching, and HTTPS enforcement. Zero cost.

11. Resend (Transactional Email)

Send welcome emails, profile nudges, notifications from talent@stlucia.studio

Week 2
⏱ ~15 minutes (includes DNS verification)
Not urgent for launch. You don't need transactional emails on Day 1. Set this up in Week 2 once sign-ups are flowing and you want to start sending welcome emails and profile completion nudges.
  1. Go to resend.com/signup and create a free account (100 emails/day free)
  2. Go to Domains → click Add Domain
  3. Enter: stlucia.studio
  4. Resend will show you DNS records to add. Go to your Cloudflare DNS dashboard and add each record:
    Record 1 (SPF): Type: TXT | Name: @ | Value: (provided by Resend) Record 2 (DKIM): Type: CNAME | Name: resend._domainkey | Value: (provided by Resend) Record 3 (DMARC, optional but recommended): Type: TXT | Name: _dmarc | Value: v=DMARC1; p=none
  5. Back in Resend, click Verify — it may take a few minutes for DNS to propagate
  6. Once verified, go to API Keys → create a new key with Sending access permission
  7. Copy the API key (starts with re_). You'll store this in Supabase Edge Function environment variables, never in frontend code.
  8. Go to Supabase Dashboard → Edge Functions → Secrets → add: RESEND_API_KEY = re_your_key_here
Save this: Your Resend API key: re________________ (store securely, never in frontend code)
Sending from: Once configured, you can send emails from talent@stlucia.studio. Your welcome emails, nudges, and notifications will come from this address.

13. Stripe (Payments)

Employer subscription billing — $99/249/499 per month tiers

Phase 2 (Week 8+)
⏱ ~20 minutes (+ identity verification may take 1-2 days)
Don't set this up until Week 8. Stripe is only needed when you start charging employers. Focus on collecting talent profiles first. When you're ready:
  1. Go to dashboard.stripe.com/register and create an account
  2. Complete identity verification (ID + business info — may take 1-2 business days)
  3. Go to Products and create 3 products:
    Starter — $99/mo (annual: $999/yr) Professional — $249/mo (annual: $2,499/yr) Enterprise — $499/mo (annual: $4,999/yr)
  4. For each product, add a monthly recurring price and an annual recurring price
  5. Go to Developers → API Keys and copy your:
    • Publishable key (starts with pk_) — safe for frontend
    • Secret key (starts with sk_) — Supabase Edge Function only, NEVER in frontend
  6. Go to Developers → Webhooks → Add endpoint:
    Endpoint URL: https://edybhgkuttsyoouizwcw.supabase.co/functions/v1/stripe-webhook Events: checkout.session.completed, customer.subscription.updated, customer.subscription.deleted
  7. Copy the webhook signing secret (starts with whsec_)
  8. Add both keys to Supabase Edge Function secrets:
    STRIPE_SECRET_KEY = sk_live_... STRIPE_WEBHOOK_SECRET = whsec_...
