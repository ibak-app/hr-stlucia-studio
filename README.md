# Talent by St. Lucia Studio

Mobile-first HR platform for Saint Lucia talent. Collects text and video resumes.

**Live:** https://talent.stlucia.studio

## Quick Start

```bash
# Local dev
python -m http.server 8080

# Open
http://localhost:8080
```

## Docs

- `CLAUDE.md` — Main agent instructions (start here)
- `MARKETING.md` — Marketing playbook and ad templates
- `ROADMAP.md` — Product roadmap and phases
- `AI-SERVICES.md` — AI service configuration and prompts

## Stack

- Frontend: Vanilla HTML/CSS/JS (PWA)
- Backend: Supabase (PostgreSQL + Auth + Storage)
- Hosting: GitHub Pages
- Domain: talent.stlucia.studio
